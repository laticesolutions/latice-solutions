## 📐 Usage Guidelines

### **Minimum Sizes**
- **Full Logo:** 120px width minimum
- **Horizontal Logo:** 100px width minimum
- **Icon Only:** 24px × 24px minimum
- **Monogram:** 32px × 32px minimum

### **Clear Space**
Maintain clear space around logo equal to the height of the icon

### **Color Variations**

**On White Background:**
- Use full color versions (as provided)

**On Dark Background:**
- Replace dark gray text with white
- Keep blue icon as-is

**Single Color (if needed):**
- All elements in #007AFF
- Or all elements in white (on dark backgrounds)

## 🔧 How to Use in Website

### **Update Navigation Logo**

Replace the current SVG in `index.html` and `index-es.html`:

```html
<!-- Current -->
<a href="#home" class="nav-logo">
    <svg width="32" height="32">...</svg>
    <span>Latice Solutions</span>
</a>

<!-- Replace with -->
<a href="#home" class="nav-logo">
    <img src="images/logo-horizontal.svg" alt="Latice Solutions" height="32">
</a>
```

### **Update Favicon**

Replace `images/favicon.svg` with `images/logo-monogram.svg`:

```html
<link rel="icon" type="image/svg+xml" href="images/logo-monogram.svg">
```
